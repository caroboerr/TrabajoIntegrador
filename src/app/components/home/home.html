Trabajo Final Integrador

Desarrollo en Angular - Aplicación de Gestión de Usuarios y Productos

Objetivo

Construir una aplicación web utilizando Angular que permita administrar usuarios y productos mediante módulos independientes, navegación por rutas y carga diferida (Lazy Loading).

Características implementadas

  • ✅ Navegación mediante Angular Router.
  • ✅ Sidebar permanente para acceso a los módulos.
  • ✅ Lazy Loading para Usuarios y Productos.
  • ✅ Listado de usuarios y productos.
  • ✅ Visualización de detalle.
  • ✅ Alta y eliminación de registros.
  • ✅ Uso de componentes standalone.
  • ✅ Uso de Pipes y Directivas modernas (@if y @for).

Rutas principales

Ruta Descripción
/ Página de inicio
/usuarios Administración de usuarios
/usuarios/:userId Detalle de usuario
/usuarios/alta Alta de usuario
/productos Administración de productos
/productos/:id Detalle de producto

Despliegue

La aplicación debe desplegarse en un servicio de hosting como GitHub Pages, Vercel, Netlify o similares.

Se recomienda utilizar Vercel, ya que es la plataforma utilizada durante el desarrollo de la materia.

Formato de Presentación

  • ✅Repositorio en GitHub con el código fuente completo del proyecto.
  • ✅Enlace a la aplicación desplegada para su revisión y validación.
  • ✅Documentación clara y actualizada en el archivo README.md.
  • ✅Organización, legibilidad y limpieza del código.
